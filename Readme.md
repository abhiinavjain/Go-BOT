# Resolve AI

A stateful, database-backed AI chatbot API built in Go (Golang). This project demonstrates a professional-grade backend architecture, managing conversation memory and AI logic.

---
##  Tech Stack

* **Backend:** Go (Golang)
* **API Router:** `chi`
* **Database:** PostgreSQL
* **Go DB Driver:** `pgx`
* **AI:** Google Gemini API
* **Local DB:** Docker
